## What I do

- Create and manage tmux sessions, windows, and panes
- Configure tmux with custom key bindings and status bars
- Write and maintain .tmux.conf configurations
- Set up automated tmux layouts for specific project types
- Troubleshoot tmux session and connectivity issues

## When to use me

Use this skill when:

- Setting up or modifying tmux configuration files
- Creating terminal session automation scripts
- Configuring development environments with tmux
- Building multi-pane terminal layouts for projects

## Best practices

- Use descriptive session names that reflect project or task context
- Prefer script-based automation over manual session management
- Document custom key bindings and workflows clearly
- Test configurations across different terminal emulators when possible
